## 📌 Key wins

- Reduced ML model **time-to-market by 10–15%** — built a self-service feature store platform (1k–5k features, auto-generated marts, no manual DAGs)
- Migrated Feature Store from **Greenplum → Iceberg + Trino + S3** end-to-end: storage model, historization, versioning, all pipelines
- Cut data service **latency from seconds → 100–400 ms**
- Reduced legacy **DAG execution time from 15–30 min → 3–8 min**
- Improved ClickHouse heavy queries by **30–100%** via sharding and replication tuning
- Migrated **1,000+ data marts** from Oracle to Greenplum, rewrote ~500 Informatica ETL flows to Airflow
